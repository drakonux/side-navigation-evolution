# Side Navigation Evolution

Prototype exploring a redesign of the side navigation pattern in [Lexicon](https://liferay.design/lexicon/) / [Clay](https://www.clayui.com).

Built on top of [Vanilla Lexicon](https://marcoscv-work.github.io/lexicon-vanilla/) by [@marcoscv-work](https://github.com/marcoscv-work) — a zero-dependency HTML/CSS implementation of the Lexicon design system.

**Live preview:** https://drakonux.github.io/side-navigation-evolution/

---

## Files

| File | Purpose |
|------|---------|
| `nav-prototype.html` | Main prototype — side navigation explorations |
| `document-manager.html` | Supporting prototype: Documents & Media app |
| `components.html` | Component reference from the base kit |
| `tokens.css` | Design tokens (light mode) |
| `tokens-dark.css` | Dark mode overrides |
| `tokens-high-contrast.css` | High-contrast overrides |
| `components.css` | Component styles |
| `icons.svg` | Clay icon sprite |

---

## Base kit

This prototype uses [Vanilla Lexicon](https://marcoscv-work.github.io/lexicon-vanilla/) as its foundation.
No build step, no npm — open any HTML file directly in a browser.
